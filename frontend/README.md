# Nexus AI Agent UI

Minimal React frontend for submitting a goal and viewing agent output.

## Quick Start

```bash
npm install
npm start
```

Runs at `http://localhost:3000` and sends requests to backend `http://127.0.0.1:8000/run-agent`.

## Key File

- `src/App.js`: input box, run button, API call, and JSON result rendering.

## Scripts

- `npm start`: run development server
- `npm test`: run tests
- `npm run build`: production build
